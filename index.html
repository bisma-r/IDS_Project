How it works

Data Science Pipeline

Each stage of the standard DS pipeline maps directly to a Python script in the project.

01
🎯
Problem Definition
01_download_data.py
Define target event classes: Foul and Yellow card. Penalty and Red card excluded due to insufficient training samples.
02
📥
Data Collection
build_index_only.py
Download SoccerNet annotations and build event_index.csv mapping events to video timestamps.
03
🎬
Preprocessing
02_extract_features.py
Extract ±5s frame windows per event. Run YOLOv8 to detect players (class 0) and ball (class 32).
04
⚙️
Feature Engineering
utils.py → compute_features()
Compute 14 spatial and temporal features: wall formation index, ball-to-spot distance, optical flow, and more.
05
📊
EDA & Modelling
03_train_model.py
Explore class distribution (509 fouls, 90 yellow cards). Apply SMOTE. Train Random Forest (200 trees) with 5-fold cross-validation.
06
🚀
Deployment
04_predict.py
Sliding window scan with temporal smoothing and cooldown deduplication. Outputs timestamped event log.
